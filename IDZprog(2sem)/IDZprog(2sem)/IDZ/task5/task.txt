Написать программу для проверки попадания введенных координат (x, y) исследуемой точки в область объединения трех фигур:
 прямоугольник (9;10), (9;13), (18;13), (18;10); круг (8;7), r=4; треугольник (11;2), (11;5), (14;3).
Пользователь должен вводить координаты точки int x, int y, например: 4, 0; пример вывода: yes, no.
